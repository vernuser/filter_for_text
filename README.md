
# 恶意信息过滤系统

- 默认账号密码是admmin/admin123

一个课程设计项目，主要由 AI 辅助完成实现与优化，核心功能是对文本、URL、IP 等内容进行安全分析与过滤，并提供自学习能力持续提升识别效果。

## 功能概览
- 内容分析：统一接口对文本/URL/IP进行恶意性判定
- 机器学习：文本采用 TF‑IDF 特征，模型在随机森林、SVM、朴素贝叶斯间自动择优；URL 使用手工特征 + 随机森林
- 自学习：预测结果与用户反馈回灌训练样本与特征库
- Web 界面：登录、仪表盘、集成测试页 `/smoke`，支持自定义输入测试
- 数据库兼容：支持 SQLite/MySQL，包含去重与迁移逻辑

## 快速开始
```bash
python run_fairy_ui.py
# 可选：设置环境变量
# $env:FAIRY_HOST='127.0.0.1'; $env:FAIRY_PORT='8000'; $env:FAIRY_DEBUG='1'
```
访问 `http://127.0.0.1:8000/` 登录并进入仪表盘，再进入 `/smoke` 页面运行集成测试或进行自定义分析。

## 目录结构
- `core/` 数据库管理、黑名单、通用后端模块
- `ml/` 机器学习引擎与特征提取、训练/预测
- `ui/` Flask 应用、模板与静态资源
- `scripts/` 集成测试脚本等
- `security/` 安全与完整性检查
- `extensions/` 扩展能力（时间控制等）
