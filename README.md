# 网络内容安全过滤系统 (Network Content Security Filter System)

## 项目简介
这是一个基于 Python 的网络内容安全过滤系统，集成了网页过滤、文件过滤、黑名单管理、时间控制与安全防护等功能。系统采用模块化设计，支持规则自定义与扩展，适用于网络安全课程设计与研究。

## 主要功能

*   **网页内容过滤**：拦截恶意网址、钓鱼网站及不当内容。
*   **文件安全扫描**：支持多种格式文件的安全检测。
*   **智能特征识别**：基于特征库的内容匹配与判定。
*   **黑名单管理**：支持黑名单的自动更新与手动维护。
*   **访问控制**：基于角色的用户权限管理。
*   **时间控制**：灵活的上网时间管理策略。
*   **系统安全**：文件完整性校验与防篡改保护。

## 目录结构

```
filter_for_text/
├── core/                  # 核心组件（过滤引擎、黑名单、访问控制）
├── ml/                    # 智能分析模块
├── ui/                    # Web 管理界面
├── data/                  # 数据库与配置文件
├── config/                # 系统配置
└── modules/               # 功能模块
```

## 运行指南

### 1. 环境准备
确保已安装 Python 3.8+ 环境，并安装所需依赖：
```bash
pip install -r requirements.txt
```

### 2. 启动系统
运行以下命令启动 Web 管理界面：
```bash
python -m ui.fairy_web
```
默认访问地址：http://127.0.0.1:8000
默认管理员账号：`admin` / `admin123`

## 开发说明

本系统仅供学习与研究使用。

### 接口说明
系统提供统一的分析接口 `/api/analyze`，支持文本、URL、IP 等多种类型的安全检测。

### 注意事项
*   系统首次运行会自动初始化数据库与配置文件。
*   黑名单更新可能需要一定时间，建议在网络状况良好时进行。
